# 🛒 Retail Sales Intelligence & 15-Day Forecast Dashboard

An end-to-end Business Intelligence and Sales Forecasting solution built
using Power BI to analyze retail performance and generate short-term
sales predictions.

This project transforms raw transactional data into actionable insights
for data-driven decision-making.

------------------------------------------------------------------------

## 📌 About the Project

This dashboard provides a comprehensive analysis of retail sales
performance across regions, states, product categories, customer
segments, shipping modes, and payment types.

It combines descriptive analytics with a 15-day time-series sales
forecast, enabling short-term demand planning and performance
monitoring. The solution demonstrates the complete BI workflow --- from
data cleaning and modeling to KPI design and predictive insights.

## 🚀 Dashboard Features

### 🔹 Sales Performance Analysis

-   Regional sales distribution (Central, East, South, West)
-   State-level performance insights
-   Category & sub-category analysis
-   Customer segment breakdown

### 🔹 Operational Insights

-   Shipping mode analysis (Standard, Second, First Class, Same Day)
-   Payment mode distribution (COD, Online, Cards)
-   Delivery performance tracking

### 🔹 Trend & Growth Analysis

-   Monthly sales trends
-   Year-over-Year comparison (2019 vs 2020)
-   Seasonal performance insights

### 🔹 Predictive Forecasting

-   15-Day Sales Forecast
-   Time-series trend projection
-   Short-term business planning support
